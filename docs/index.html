🛰️ XHRay - Telemetry Correlator

XHRay is a modular Tampermonkey userscript that captures and correlates browser telemetry events — including click, submit, xhr, fetch, WebSocket, and sendBeacon — with live rule matching and a draggable UI overlay panel.

🔧 Key Features

  • DOM + network telemetry capture
  • Live rule matching engine
  • Draggable, toggleable floating panel
  • Built-in JSON rule editor
  • WebSocket + Beacon tracking
  • Dark mode, log size, persistence options

📦 Installation

  1. Install Tampermonkey
  2. Install XHRay Userscript
